Dependencies:
connect-mongo
dotenv
express
mongodb
mongoose
nodemon

Install script:
npm install connect-mongo dotenv express mongodb mongoose --save //Installs all the necessary dependancies
npm install nodemon --save-dev //Installs nodemon as a developer dependancy

How to run
type "npm run dev"
